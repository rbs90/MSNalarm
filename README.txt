This program is for sending one message to all contacts in a contactlist of your msn-server account (also when they are offline).

Syntax: java -jar msnAlarm.jar "Das hier ist der Text"

You need a account.txt file, first row is the username, secound the password of your account.



Dieses Programm dient zum verbreiten einer nachricht per MSN.

Syntax: java -jar msnAlarm.jar "Das hier ist der Text"


Es wird eine account.txt-Datei benötigt, diese muss in der ersten Zeile den Benutzernamen und in der zweiten dessen Passwort enthalten.

Die Empfänger der Nachricht sind alle in der Kontaktliste enthaltenen Kontakte, egal ob diese gerade online sind.
